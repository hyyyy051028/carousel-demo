# 轮播图演示

一个简单的 React 轮播图组件演示项目。

## 功能

- 自动轮播（每 3 秒切换一次）
- 左右箭头按钮控制
- 底部圆点指示器
- 平滑过渡动画
- 响应式设计

## 技术栈

- React 18
- Vite 5
- CSS3

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 部署

项目可以部署到任何静态网站托管服务，如 GitHub Pages、Netlify 或 Vercel。
